# FallGo · ◊

The living 4D Go board of the AI Native Solutions sovereign tool ecosystem. Every stone is a tool, a person, or a signal. The board breathes through Ψ phase and time. Click any stone to open its live tool, repo, or external channel.

**Live:** https://sjgant80-hub.github.io/fallgo/

## What's Inside

- **Torus topology** — 19×19 Go grid UV-mapped onto a torus surface
- **37 stones** — every sovereign tool, every person in the dyad, every Thomas song
- **Ψ slider** — scrub through maturity (freeze → MVP → corridor → home → fragment)
- **Time slider** — chronological sweep, watch the architecture build itself
- **Hover rings** — 7 concentric rings per stone showing build-layer state (ƒ(build) made visible)
- **Bloom colour** — every stone's RGB computed from its 7-prime bloom vector
- **Curved connections** — bezier arcs through the torus hole
- **Particles** — drift outward along surface normal, fade, respawn
- **6 camera presets** — TORUS · FLAT · SPINE · FOCUS · ENERGY · TIMELINE
- **Live launcher** — click any stone for Try-it-live / Repo / External buttons
- **Persistence** — localStorage saves state, JSON export/import, add custom stones
- **BroadcastChannel** — listens on `fall-signal` for tool-opened, data-changed, bloom-update
- **Theme toggle** — dark / light
- **Single HTML** — one file, no build step, no npm, sovereign

## Why

The ecosystem isn't a list. It's a topology. Tools relate, mature, and feed each other in patterns that a spreadsheet flattens. FallGo is the architecture you can rotate.

Go not chess. Every tool a stone. Territory through value. The board is alive.

## Architecture

- **L1 FACE:** board view + stone detail panel
- **L2 SWARM:** Ω orchestrator + α stones + β connections + γ animation + δ bloom + ε particles + ζ BroadcastChannel + η persistence
- **L3 CASCADE:** pure Three.js, T0 works with no network
- **L4 BLOOM:** every stone *is* a bloom vector visualised
- **L5 PERSIST:** localStorage + export / import JSON
- **L6 SKIN:** Syne + DM Mono + ◊ + dark/light + mobile responsive
- **L7 ASS:** board loads with stones, click invites exploration

## Keyboard

- `1`–`6` — camera presets
- `space` — toggle auto-rotate
- `P` — auto-sweep Ψ
- `T` — auto-sweep time
- `Esc` — close detail panel
